# 統計学学習ツール (Stats Learning Tool)

統計学の学習、特に確率分布の定義や性質の暗記を助けるためのインタラクティブな学習ツールです。

## 概要
HTMLベースの静的ウェブアプリとして動作し、確率分布の性質を「思い出す練習（フラッシュカード）」や、事前作成されたクイズによる学習が可能です。

## 特徴
- **公式の暗記練習**: 表形式やカード形式で数式を表示。
- **動的な目隠し機能**: クリックで正解（平均・分散・MGFなど）を表示し、自己クイズが可能。
- **クイズ機能**: 静的なクイズデータに基づき、選択問題をランダムに提示。
- **ポータビリティ**: 静的なHTML/JS構成のため、HTTPサーバーに置くだけでスマホ等からも利用可能。

## ディレクトリ構成
```text
/
├── index.html         # 学習項目一覧（トップ画面）
├── study.html         # 学習画面（フラッシュカード形式）
├── quiz.html          # クイズ画面
├── style.css          # 共通スタイルシート
├── data/
│   ├── items.js       # 確率分布・数学関数のデータ定義
│   └── quizzes.js     # 静的なクイズ問題データ
├── design.md          # 技術的設計・データ構造
├── requirements.md    # 要件・制約
├── todo.md            # タスク進捗管理
├── items.md           # 学習対象の項目リスト
├── README.md          # 本ファイル
└── GEMINI.md          # AI専用コンテキスト（進捗・命令）
```

## 使い方
1.  `index.html` をブラウザで開く（ローカルファイルまたはHTTPサーバー経由）。
2.  学習したい項目を選択。
3.  「学習する」でフラッシュカード形式の想起練習、「クイズ」で選択問題に挑戦。

詳細は `[design.md](design.md)` および `[requirements.md](requirements.md)` を参照してください。

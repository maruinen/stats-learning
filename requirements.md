# 要件定義: 統計学学習ツール

## 1. 目的
統計学の主要な確率分布や定理の性質を効率的に学習・記憶することを支援する。ユーザーが能動的に知識を想起し、定着させるためのインタラクティブなWebアプリケーションを提供する。

## 2. 機能要件

### 2.1. 学習項目 (Items)
- **FR-01: 一覧表示**: 以下の学習項目を一覧で表示できること。
    - 代表的な離散型・連続型確率分布
    - 統計学の主要な定理（大数の法則、中心極限定理など）
    - 関連する数学的関数・概念（ガンマ関数、標準誤差など）
- **FR-02: 詳細表示**: 各項目の詳細な定義を確認できること。
    - 確率（密度）関数
    - 期待値、分散
    - 確率母関数（離散型）またはモーメント母関数（連続型）
- **FR-03: フラッシュカード機能**:
    - 学習画面では、デフォルトで数式などの詳細情報が隠されていること。
    - ユーザーのアクション（クリック等）により、隠された情報を表示できること。

### 2.2. クイズ機能 (Quiz)
- **FR-04: 静的クイズ**: 各学習項目に対して、事前に作成された複数のクイズが出題されること。
- **FR-05: ランダム出題**: 同じ項目のクイズは、毎回ランダムな順序で出題されること。
- **FR-06: 選択式**: クイズは4択の選択式であること。
- **FR-07: 正誤判定と解説**: 解答後、すぐに正解・不正解が表示され、解説が確認できること。
- **FR-08: ループ機能**: ある項目のクイズをすべて解き終えたら、再度シャッフルされて最初から挑戦できること。

## 3. 非機能要件

- **NFR-01: ポータビリティ**:
    - 静的なHTML/CSS/JavaScriptで構成され、特定のバックエンドを必要としないこと。
    - ローカル環境でも、一般的なHTTPサーバー上でも動作すること。
- **NFR-02: レスポンシブ**: PCおよびスマートフォンの両方の画面サイズで快適に利用できること。
- **NFR-03: 数式表示**: 数式は MathJax を用いて、高品質にレンダリングされること。

## 4. 将来的な拡張（現時点では対象外）
- Gemini API を利用した動的なクイズ生成機能。
- 学習履歴のローカル保存と、それに基づいた出題最適化。

# デザインドキュメント: 統計学学習ツール (Stats Learning Tool)

## 1. 目的
統計検定等の学習を効率化するため、主要な確率分布の性質（平均、分散、モーメント母関数など）を「思い出す練習」ができ、人間が作成したクイズで理解度を確認できるウェブアプリケーションを構築する。

## 2. ターゲット
- 確率分布の公式を暗記・理解したい学習者。
- 数式を確認するだけでなく、能動的にアウトプット（クイズ）を行いたいユーザー。

## 3. 主要機能
- **分布一覧表示**: `items.md` にリストされた確率分布や関数のカタログ。
- **フラッシュカード機能**:
    - デフォルトで数式を隠し、クリック/ボタン操作で表示する（想起練習）。
- **クイズ機能**:
    - あらかじめ人間が作成した静的なクイズデータ (`data/quizzes.js`) を使用。
    - 各項目（分布など）に関連付けられた4択問題をランダムに出題。
    - 正誤判定と解説の表示。
    - 将来的な拡張として Gemini API による動的生成も検討。
- **データ管理**:
    - 数式や定義を `data/items.js` で管理。
    - クイズの問題文と選択肢を `data/quizzes.js` で管理。

## 4. 技術スタック
- **フロントエンド**: HTML5, Vanilla CSS, Vanilla JavaScript (ES6+)。
- **データ形式**: JavaScript (CORS回避のため定数として管理)。
- **数式表示**: MathJax。

## 5. 画面遷移 / 構成
1.  **トップ画面 (`index.html`)**: 学習項目の選択。
2.  **学習画面 (`study.html`)**: 選択した項目の詳細表示（隠蔽/表示切り替え）。
3.  **クイズ画面 (`quiz.html`)**: 作成済み問題の提供と回答。

## 6. データ構造案

### 学習項目 (`data/items.js`)
ブラウザのセキュリティ制限（CORS）を回避し、ローカル環境やモバイル環境で直接ファイルを開いても動作するように、JavaScriptの定数として管理する。

```javascript
const STATS_DATA = { ... };
```

### クイズデータ (`data/quizzes.js`)
人間が作成した特定の問題を管理する。

```javascript
const QUIZ_DATA = [
  {
    "id": "q1",
    "itemId": "binomial",
    "question": "二項分布 B(n, p) の分散として正しいものはどれか？",
    "options": ["np", "np(1-p)", "np^2", "p(1-p)"],
    "answerIndex": 1,
    "explanation": "二項分布の分散は npq (q=1-p) です。"
  }
];
```
